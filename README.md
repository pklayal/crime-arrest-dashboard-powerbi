# 🚔 Crime & Arrest Insights Dashboard (2010–2019)

## 📊 Overview
This project analyzes arrest data from 2010–2019 using Power BI to uncover crime trends, hotspots, and demographic patterns.

---

## 📁 Dataset
- Source: Data.gov
- Records: ~1.32 Million
- Columns: 25

📌 Note: Dataset not included due to large size  
👉 Download here:  
https://catalog.data.gov/dataset/arrest-data-from-2010-to-2019

---

## 📌 Key KPIs
- Total Arrests
- Average Age
- Male %
- Top Year

---

## 📈 Features
- Yearly & Monthly Trends
- Top 5 Areas by Arrests
- Top Crime Categories
- Arrests by Hour
- Gender Distribution
- Crime Hotspots Map

## 💡 Key Insights
- Declining trend in arrests over years
- Certain areas show consistently high crime
- Majority arrests are male (~79%)
- Specific hours show higher crime activity

## 🚀 Project Outcome
This dashboard transforms raw data into actionable insights using interactive visualizations and supports data-driven decision making.
